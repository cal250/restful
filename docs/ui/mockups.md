# UI Mockups and Component Specification

The implemented React frontend is the canonical interactive mockup.

## Layout

- Dark `#17202a` sidebar with role-aware navigation
- White top bar with current user and logout action
- Light `#f5f7fa` workspace background
- White cards with subtle borders, rounded corners, and restrained shadows
- Fire-safety red `#c5281c` for primary actions and active navigation

## Screens

- Login
- Dashboard
- Fire Extinguishers
- Inspections
- Maintenance
- Reports
- Notifications
- Users
- Profile

Reusable components include `AppShell`, `PageHeader`, `MetricCard`,
`StatusBadge`, and `DataTable`.
